

### 图片懒加载
```html
<!DOCTYPE html>
<html>
<head>
	<script src="https://unpkg.com/scrollreveal@4"></script>
</head>
<body>

	<div class="headline">
		Widget, Inc.
	</div>

	<script>
		ScrollReveal().reveal('.headline');
	</script>
</body>
</html>
```
